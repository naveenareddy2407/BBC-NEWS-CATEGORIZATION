# 📰 BBC News Categorization using Machine Learning & NLP

## 📌 Overview

This project is an end-to-end **Machine Learning and Natural Language Processing (NLP)** application that automatically classifies BBC news articles into predefined categories.

The project focuses on improving text representation using **GloVe word embeddings** and evaluates multiple machine learning algorithms to identify the most accurate classifier.

---

## 🚀 Features

- Automatic news article classification
- Text preprocessing and cleaning
- NLP-based feature extraction using **GloVe Embeddings**
- Comparison of multiple Machine Learning models
- Performance evaluation using accuracy, confusion matrix, ROC curve, and AUC score
- Visualization of model performance

---

## 🛠️ Tech Stack

- **Programming Language:** Python
- **Libraries:** NumPy, Pandas, Scikit-learn, NLTK, Matplotlib, Seaborn
- **NLP:** GloVe Word Embeddings
- **Machine Learning:** K-Nearest Neighbors (KNN), Decision Tree, Random Forest
- **Development Environment:** Jupyter Notebook / VS Code

---

## 📂 Dataset

The project uses the **BBC News Dataset**, which contains labeled news articles belonging to multiple categories such as:

- Business
- Entertainment
- Politics
- Sport
- Technology

---

## ⚙️ Project Workflow

1. Load BBC News Dataset
2. Data Cleaning
3. Text Preprocessing
   - Lowercasing
   - Tokenization
   - Stopword Removal
   - Lemmatization
4. Feature Extraction using GloVe Embeddings
5. Train Multiple Machine Learning Models
6. Evaluate Model Performance
7. Compare Results
8. Select Best Performing Model

---

## 🤖 Models Evaluated

- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

---

## 📊 Evaluation Metrics

The models were evaluated using:

- Accuracy
- Confusion Matrix
- ROC Curve
- AUC Score

---

## 🏆 Results

Among all the evaluated models, **K-Nearest Neighbors (KNN)** achieved the best overall performance on the BBC News Dataset.

The use of **GloVe word embeddings** improved semantic understanding compared to traditional word-frequency-based approaches.

For detailed visualizations and outputs,see the **Results** folder
